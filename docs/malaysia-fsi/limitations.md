# Limitations

## Implemented

- Maybank CSV parsing
- Offline invoice matching
- Offline reconciliation reporting

## Not Implemented

- MyInvois submission/integration
- SSM verification
- JPN verification
- BNM verification
- CTOS integration
- Bursa integration
- Live bank APIs

## Decision Boundaries

This system is assistance-only. It must not be used as final tax, legal, accounting, AML/CFT, or regulatory decision authority.

**HUMAN REVIEW REQUIRED** for every workflow output.
